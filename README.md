# AI Study & Exam Preparation Coach


Multi-agent system that automates study planning, revision scheduling and doubt solving.


## Overview
This repository contains a modular implementation of a multi-agent study coach. The code is educational and designed for the Kaggle Capstone submission (Concierge Track).


## Features
- Planner Agent: builds a personalized study schedule
- Doubt Solver Agent: answers student questions using an LLM (placeholder)
- Revision Loop Agent: daily revision recommendations
- Memory Agent: tracks weak topics and progress
- Resource Agent: fetches external materials via a search tool
